# Atik Shahrear Ananto — Premium Portfolio 2026

A modern, animated portfolio website built with **Next.js**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **GSAP**.

## What is included

- Futuristic dark visual direction with animated gradient/particle background
- Framer Motion hero animations and floating UI elements
- GSAP ScrollTrigger reveal effects for each section
- Custom cursor aura for desktop
- Magnetic CTA buttons
- Responsive navbar with mobile menu
- Skill marquee and grouped skill cards
- Premium project cards with glow-follow hover effects
- Experience, education, and leadership timeline
- Contact section with email, phone, GitHub, and CV download
- Your CV placed in `/public/resume-atik-shahrear-ananto.pdf`

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build for production

```bash
npm run build
npm run start
```

## Customize

Most portfolio content lives in:

```bash
src/data/portfolio.ts
```

Edit that file to change your skills, projects, contact info, timeline, and services.

## Deploy

Recommended deployment platforms:

- Vercel
- Netlify
- Firebase Hosting with Next adapter

For the easiest deployment, push this project to GitHub and import it into Vercel.
