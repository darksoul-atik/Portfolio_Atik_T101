# Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

## Navigate to the Project

```bash
cd your-repository
```

## Install Dependencies

```bash
npm install
```

---

# Run Locally

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

# Production Build

```bash
npm run build
npm run start
```

# Resume Update

The downloadable CV is stored inside:

```bash
public/resume-atik-shahrear-ananto.pdf
```

To update your resume:

1. Replace the old PDF
2. Keep the same filename

No additional code changes are required.
